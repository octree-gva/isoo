# ISOO documentation site

Docusaurus site for operators and infrastructure teams.

Published at **https://voca.github.io/isoo/** (GitHub Pages via `.github/workflows/website.yml`).

```bash
npm install
npm start      # dev server — http://localhost:3000/isoo/
npm run build  # static output in build/
```

`baseUrl` is `/isoo/` for project-site hosting under `voca.github.io`.
